# 归档说明

此目录包含已废弃的代码文件，被新的预计算架构替代。

## 归档文件

### DailyPriceInitializer.cs
- **归档日期**: 2025-11-27
- **替代者**: `DailyMarketOpener.cs`
- **原因**: 重构为预计算架构，日初价格不再实时生成，而是从预计算数据读取

### IntradayNewsService.cs
- **归档日期**: 2025-11-27
- **替代者**: `NewsSchedulePlayer.cs`
- **原因**: 重构为预计算架构，盘中新闻不再随机生成，而是按预定时间表播放

## 架构变更说明

**旧架构**（动态生成）：
- 每日开盘时实时计算价格
- 盘中随机生成新闻事件

**新架构**（预计算）：
- 季节开始时预计算整季28天的价格轨迹和新闻事件
- 运行时仅播放预计算数据
- 好处：可预测、性能好、易测试、支持存档
